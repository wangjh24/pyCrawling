# 회사 주식 정보 크롤링 
> 네이버 증권에서 회사코드를 통해 종합정보, 투자자 매매동향, 뉴스 , 토론 내용을 크롤링하여 DB저장 및 조회를 하는 프로젝트

## 1.Features
- 종합 정보 조회 기능 
- 투자자 매매 동향 조회 기능
- 뉴스 조회 기능
- 종목 토론 조회 기능

## 2.Tech Stack
- **Frontend**: React, JavaScript (예정)
- **Backend**: Python, PostgreSQL
