# 会社の株式情報 クローリング
> NAVER証券で会社コードを通じて総合情報、投資家売買動向、ニュース、討論内容をクローリングしてDB保存および照会を行うプロジェクト

## 1.Features
- 総合情報照会機能 
- 投資家売買動向照会機能
- ニュース照会機能
- 種目討論照会機能

## 2.Tech Stack
- **Frontend**: React, JavaScript (予定)
- **Backend**: Python, PostgreSQL

## 3.Getting Started
1.cmdコマンドウィンドウで会社コードを入力
2.ご希望のサービス番号を入力
